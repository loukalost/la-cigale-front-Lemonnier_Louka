# Projet La Cigale - Implémentation Homepage depuis Stitch

## Contexte
Je vais développer la homepage du site web de La Cigale (brasserie Art Nouveau à Nantes) en utilisant comme source de vérité les maquettes créées dans Google Stitch.

## Design de référence (Stitch)
- **Projet Stitch** : La Cigale - Homepage
- **Écran principal** : home_desktop
- **Écran mobile** (si créé) : home_mobile

## Instruction pour l'agent
Tu dois utiliser le serveur MCP Stitch configuré pour :
1. Récupérer les informations du projet Stitch "La Cigale - Homepage"
2. Analyser la structure et le design de l'écran "home_desktop"
3. Extraire les composants, couleurs, typographies et layout définis dans la maquette

## Sections attendues (selon la maquette)
- Hero (image + baseline + CTA réservation)
- Section Brasserie (histoire, patrimoine)
- Section Carte (plats signatures)
- Section Réservation
- Section Serveur IA
- Footer

## Stack technique
- React / Next.js (ou autre selon recommandation de l'architecte)
- Responsive mobile-first
- Respect strict du design Stitch

## Première action
Peux-tu lister les projets Stitch disponibles via MCP et récupérer les détails de l'écran "home_desktop" du projet "La Cigale - Homepage" ?
